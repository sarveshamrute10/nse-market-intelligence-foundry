# NSE Market Intelligence Platform on Palantir Foundry

## Overview

Built an end-to-end market intelligence platform using Palantir Foundry.

The solution ingests NSE Bhavcopy trading files,
implements a Medallion architecture,
models business entities using Ontology,
and exposes insights through Workshop applications.
